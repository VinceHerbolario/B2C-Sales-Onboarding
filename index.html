B2C Sales Onboarding Framework

Sales qualifies interest and treatment need first; the Medical Team is endorsed only after the presentation is done. Teleconsultation is the default consultation mode — face-to-face happens only when required, either at the patient's premises or an agreed location.

Cloud Connection

This app is pre-connected to the b2c-sales-onboarding Firebase project and talks to Firestore directly, with no sign-in step. That means your Firestore security rules must allow open read/write access — set this once in the Firebase Console under Firestore Database → Rules:

rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /{document=**} {
      allow read, write: if true;
    }
  }
}

Since this opens the database to anyone with the project config, it is meant for an internal team tool only — do not use this rule set if the file will be publicly reachable by outsiders.
